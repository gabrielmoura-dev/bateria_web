# 🥁 Bateria Web

Uma bateria virtual interativa construída com tecnologias web modernas. Toque bateria diretamente no seu navegador utilizando o teclado ou cliques do mouse!

Você pode visualizar o projeto online aqui: [Link do seu site (ex: GitHub Pages/Vercel)](#)

## ✨ Funcionalidades

- **Sons de Boa Qualidade:** Amostras de áudio realistas para cada peça da bateria.
- **Interatividade Total:** Toque clicando nos componentes ou usando atalhos do teclado.
- **Feedback Visual:** Animações ao pressionar as teclas ou clicar nos tambores/pratos.
- **Interface Responsiva:** Adaptado para diferentes tamanhos de tela.

## 🎹 Controles (Teclado)

| Tecla | Peça da Bateria |
| :---: | :--- |
| **A** | Bumbo (Kick) |
| **S** | Caixa (Snare) |
| **D** | Tom Alto |
| **F** | Tom Médio |
| **G** | Tom Baixo (Surdo) |
| **E** | Prato de Ataque (Crash) |
| **W** | Hi-hat (Chimbal) |

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido com:

- [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML) - Estrutura.
- [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS) - Estilização e Animações.
- [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) - Lógica de áudio e eventos.
